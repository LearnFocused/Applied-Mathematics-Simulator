The aim of this project is to provide simulators to aid students in learning Leaving Certificate Applied Mathematics.
Students can select from a number of scenarios for each topic and adjust relevant parameters.
It has been tested with "sane" values that would usually be found in Applied Mathematics. It is not guaranteed to perform well with large values.

A browser that has JavaScript enabled is required to use these simulators.
Tested to work with the following browsers:

  • FireFox latest version
  • Chrome latest version and also Android version
  • Safari on iOS
  • Internet Explorer 8 and later
Please click "Allow blocked content" to allow JavaScript if using Internet Explorer
